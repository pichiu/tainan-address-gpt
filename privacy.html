隱私權政策

台南地址座標查詢 GPT 尊重並保護使用者的隱私權。

1. 收集的資訊

本工具僅根據使用者輸入的地址資訊（如行政區、里、鄰、地址關鍵字等）進行查詢，這些資料不會被儲存、記錄或用於其他用途。

2. 第三方服務

地址查詢結果來自台南市政府提供的公開地址座標查詢 API，不涉及個人資料。

3. 使用者資料安全

本工具不會收集、儲存或分享任何個人身份資訊。

4. 資料來源

資料來源：臺南市政府 113年 臺南市門牌坐標資料
此開放資料依 政府資料開放授權條款（Open Government Data License） 進行公眾釋出。使用者於遵守本條款各項規定之前提下，得利用之。

5. 聯絡方式

如對本工具的隱私政策有任何疑問，請透過 GitHub Repo 中的 Issues 或 Email 聯絡我們。
